隐私政策

最后更新：2025年10月16日（🔒 极简统计版）

您的隐私对我们至关重要

家长锁 ("我们" 或 "应用") 致力于保护您的隐私。本隐私政策说明了我们收集、使用和保护您信息的方式。

1. 我们收集哪些信息

1.1 极简技术统计（🔒 默认启用，随时可关）

为了改进应用，我们会自动收集以下5项完全匿名的技术信息：

  • 应用版本号（如 1.0.5） - 用于版本对比和问题排查
  • iOS主版本号（如 iOS 17，不含具体子版本） - 用于兼容性测试
  • 功能使用标记（仅标记是/否，不计次数） - 了解哪些功能被使用
  • 崩溃次数范围（0次、1-5次、5+次） - 评估应用稳定性
  • 最后活跃周（按周统计，如 2024-W42） - 了解用户活跃情况

✓ 极简数据：只有5项基本技术信息

✓ 完全匿名：无法识别您的个人身份

✓ 默认启用：因为数据极简且完全匿名，符合合法利益原则

✓ 随时关闭：您可以在"设置" → "改善用户体验"中随时关闭

1.2 应用功能必需的信息

为了提供应用的核心功能，我们会在您的设备本地存储：

  • 您设置的时间段配置（存储在本地）
  • 您设置的时长限制（存储在本地）
  • 您设置的位置限制（存储在本地）
  • 应用的密码设置（存储在本地，使用iOS安全存储）

重要说明：这些配置信息仅存储在您的设备上，不会上传到任何服务器。

1.3 我们绝不收集（🔒 隐私保护承诺）

✗ 您的姓名、电话号码或邮箱地址

✗ 孩子的姓名、照片或任何个人身份信息

✗ 任何地理位置信息（精确坐标、位置名称、区域等）

✗ 用户配置的任何详情（配置名称、时间、时长、数量等）

✗ 精确的使用时间（不记录日期、时间或时刻）

✗ 使用频率和次数（只标记"用过/没用过"）

✗ 设备型号（不收集具体型号）

✗ iOS详细版本（只记录主版本号，如iOS 17）

✗ 设备唯一标识符（IDFA/UDID）用于追踪

✗ 应用内的任何内容、消息或数据

2. 如何使用这些信息

2.1 我们使用收集的匿名信息来：

  • 改进应用功能和用户体验
  • 分析哪些功能最受欢迎，优先优化
  • 发现和修复程序错误
  • 优化应用性能和稳定性
  • 了解用户需求，开发新功能

2.2 我们承诺永远不会：

✗ 出售您的数据给第三方

✗ 用于广告投放或推送

✗ 与其他公司共享（除非法律要求）

✗ 用于跨应用或跨网站追踪

✗ 将数据与您的真实身份关联

3. 数据存储和安全

3.1 数据存储位置

✓ 应用配置存储在您的设备本地

✓ 匿名统计数据存储在您的私有iCloud空间

✓ 使用Apple CloudKit服务（符合Apple安全标准）

✓ 只有您本人能访问您的iCloud数据

✓ 其他用户完全无法看到您的数据

3.2 数据安全措施

我们采取以下措施保护您的数据：

  • 使用Apple iCloud的端到端加密存储
  • Private Database隔离（数据完全私密）
  • 匿名用户标识（使用CloudKit系统ID，不涉及真实身份）
  • 完全匿名化和模糊化处理
  • 不收集敏感的个人身份信息
  • 定期审计收集的数据类型

4. 您的隐私权利和控制

您拥有完全的控制权：

4.1 选择权 - 完全由您决定

✓ 在应用"设置" → "改善用户体验"中：

  • 开启"帮助改进应用" = 允许收集5项匿名技术数据
  • 关闭"帮助改进应用" = 完全不收集任何数据
  • 您可以随时更改此设置

默认状态：默认启用（因为数据极简且完全匿名）。首次使用时会显示简短提示，告知您收集的内容，您可以随时在设置中关闭。

4.2 透明承诺 - 数据完全匿名

✓ 所有收集的数据都已完全匿名化

  • 无法识别您的真实身份
  • 无法关联到您个人
  • 只是统计数字和范围
  • 不包含任何个人隐私信息

4.3 数据管理 - 存储在您的iCloud

✓ 完全掌控您的数据

  • 匿名数据存储在您的私有iCloud空间
  • 您可以通过iCloud设置管理存储空间
  • 删除应用会自动清理相关数据
  • 关闭"帮助改进应用"会立即停止收集

4.4 撤回同意 - 随时可以改变主意

✓ 您可以随时撤回之前的同意

  • 在设置中关闭"帮助改进应用"即可
  • 关闭后立即停止收集新数据
  • 已收集的匿名数据会在90天后自动清理
  • 不会影响应用的正常使用

5.1 我们的承诺

  • 中国《儿童个人信息网络保护规定》

6. 数据共享和披露

6.1 我们不会共享您的数据

除以下极少数情况外，我们不会与任何第三方共享您的数据：

  • 法律要求：当法律、法规或法律程序明确要求时

  • Apple Screen Time API：用于实现应用限制功能（完全在设备本地运行）

这些都是Apple的原生服务，受Apple隐私政策保护：Apple隐私政策

7. 数据保留

我们会保留您的匿名统计数据直到：

  • 您删除应用

  • 您从iCloud设置中清除应用数据

自动清理机制：超过90天的匿名统计数据会自动从系统中删除。

8. 国际数据传输

您的数据存储在Apple iCloud中，可能会在不同国家/地区的Apple数据中心之间传输。

  • 数据已完全匿名化，无法识别个人身份
  • 符合各国数据保护法规要求

Privacy Policy

Last Updated: October 16, 2025 (🔒 Minimal Stats Edition)

Your Privacy is Our Priority

PatLock ("we" or "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information.

1. Information We Collect

1.1 Minimal Technical Statistics (🔒 Default On, Turn Off Anytime)

To improve the app, we automatically collect the following 5 items of completely anonymous technical information:

  • App version number (e.g., 1.0.5) - For version comparison and troubleshooting
  • iOS major version (e.g., iOS 17, no specific sub-version) - For compatibility testing
  • Feature usage flags (only mark yes/no, no counts) - To understand which features are used
  • Crash count range (0 times, 1-5 times, 5+ times) - To assess app stability
  • Last active week (weekly stats, e.g., 2024-W42) - To understand user activity

✓ Minimal data: Only 5 basic technical items

✓ Completely anonymous: Cannot identify your personal identity

✓ Default on: Because the data is minimal and completely anonymous, complying with legitimate interest principles

✓ Turn off anytime: You can disable this in "Settings" → "Improve Experience" at any time

1.2 Information Required for App Features

To provide core app functionality, we store locally on your device:

  • Your time slot configurations (stored locally)
  • Your duration limit settings (stored locally)
  • Your location restriction settings (stored locally)
  • App password settings (stored locally using iOS secure storage)

Important: This configuration information is only stored on your device and never uploaded to any server.

1.3 We Absolutely Do NOT Collect (🔒 Privacy Protection Commitment)

✗ Your name, phone number, or email address

✗ Children's names, photos, or any personally identifiable information

✗ Any geographic location information (precise coordinates, location names, regions, etc.)

✗ Any details of user configurations (configuration names, times, durations, quantities, etc.)

✗ Precise usage times (no dates, times, or timestamps recorded)

✗ Usage frequency and counts (only mark "used/not used")

✗ Device model (no specific model collected)

✗ Detailed iOS version (only major version, e.g., iOS 17)

✗ Device unique identifiers (IDFA/UDID) for tracking

✗ Any app content, messages, or data

2. How We Use This Information

2.1 We use collected anonymous information to:

  • Improve app features and user experience
  • Analyze which features are most popular for prioritized optimization
  • Discover and fix bugs
  • Optimize app performance and stability
  • Understand user needs and develop new features

2.2 We promise to NEVER:

✗ Sell your data to third parties

✗ Use it for advertising or marketing

✗ Share with other companies (unless legally required)

✗ Use for cross-app or cross-website tracking

✗ Link data to your real identity

3. Data Storage and Security

3.1 Data Storage Location

✓ App configurations stored locally on your device

✓ Anonymous statistics stored in your private iCloud space

✓ Uses Apple CloudKit service (complies with Apple security standards)

✓ Only you can access your iCloud data

✓ Other users cannot see your data

3.2 Security Measures

We take the following measures to protect your data:

  • Apple iCloud end-to-end encryption storage
  • Private Database isolation (data is completely private)
  • Anonymous user identification (uses CloudKit system ID, no real identity)
  • Complete anonymization and fuzzification
  • No collection of sensitive personal information
  • Regular audits of collected data types

4.2 Transparency Promise - Data is Completely Anonymous

✓ All collected data is completely anonymized

  • Contains no personal privacy information
