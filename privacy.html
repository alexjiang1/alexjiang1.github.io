隐私政策

最后更新：2025年10月15日

您的隐私对我们至关重要

PlayTimer+ ("我们" 或 "应用") 致力于保护您的隐私。本隐私政策说明了我们收集、使用和保护您信息的方式。

1. 我们收集哪些信息

1.1 自动收集的匿名信息

为了改进应用功能和用户体验，我们可能会收集以下完全匿名的信息：

  • 设备型号（如 iPhone 14 Pro）
  • 操作系统版本（如 iOS 17.0）
  • 应用版本号
  • 功能使用统计（如创建了几个时间限制、使用了哪些功能）
  • 应用崩溃和错误报告

1.2 用户提供的配置信息

当您使用应用功能时，我们可能收集：

  • 孩子年龄段（如 0-3岁、3-6岁，而非精确年龄）
  • 功能配置（如时间限制的设置、时间段配置）
  • 购买状态（是否为高级用户）

1.3 我们不会收集

✗ 您的姓名、电话号码或邮箱地址

✗ 孩子的姓名、照片或任何个人身份信息

✗ 具体的地理位置坐标

✗ 应用内的具体内容、消息或聊天记录

✗ 设备唯一标识符（IDFA/UDID）用于追踪

2. 如何使用这些信息

2.1 我们使用收集的信息来：

  • 改进应用功能和用户体验
  • 分析哪些功能最受欢迎，优先优化
  • 发现和修复程序错误
  • 优化应用性能和稳定性
  • 了解用户需求，开发新功能

2.2 我们承诺永远不会：

✗ 出售您的数据给第三方

✗ 用于广告投放或推送

✗ 与其他公司共享（除非法律要求）

✗ 用于跨应用或跨网站追踪

3. 数据存储和安全

3.1 数据存储位置

✓ 所有数据存储在您的私有iCloud空间

✓ 使用Apple CloudKit服务（符合Apple安全标准）

✓ 只有您本人能访问这些数据

✓ 其他用户完全无法看到您的数据

3.2 数据安全措施

我们采取以下措施保护您的数据：

  • 使用Apple iCloud的端到端加密存储
  • Private Database隔离（数据完全私密）
  • 匿名用户标识（使用CloudKit系统ID，不涉及真实身份）
  • 不收集敏感的个人身份信息
  • 定期审计收集的数据类型

4. 您的隐私权利

您拥有以下权利，并且可以随时行使：

4.1 选择权

✓ 随时开启或关闭数据收集

在应用"设置" → "隐私与数据"中，您可以随时关闭匿名统计功能。

4.2 知情权

✓ 查看已收集的数据

您可以在应用中查看我们收集了哪些关于您的数据（功能开发中）。

4.3 删除权

✓ 删除所有已收集的数据

您可以要求我们删除所有已收集的数据（功能开发中），或者直接从您的iCloud中删除。

4.4 撤回同意

✓ 随时撤回之前的同意

您可以随时在设置中关闭数据收集，我们将立即停止收集。

5. 儿童隐私保护

PlayTimer+ 是一款儿童管理应用，我们特别重视儿童隐私：

5.1 我们的承诺

✓ 不直接收集儿童的个人信息

✓ 只收集年龄段（如0-3岁），不收集精确年龄

✓ 不收集儿童的姓名、照片或任何可识别身份的信息

✓ 所有功能需要家长设置和控制

5.2 符合儿童隐私法规

我们的数据收集实践符合：

  • 美国《儿童在线隐私保护法》(COPPA)
  • 欧盟《通用数据保护条例》(GDPR) 中关于儿童的特别规定
  • 中国《儿童个人信息网络保护规定》

6. 数据共享和披露

6.1 我们不会共享您的数据

除以下情况外，我们不会与任何第三方共享您的数据：

  • 法律要求：当法律、法规或法律程序要求时
  • 保护权利：为保护我们的合法权利、财产或安全时
  • 征得同意：在征得您明确同意的情况下

6.2 第三方服务

我们使用以下第三方服务，它们都有严格的隐私政策：

  • Apple iCloud/CloudKit：用于数据存储（隐私政策：Apple隐私政策）

7. 数据保留

我们会保留您的数据直到：

  • 您选择删除数据
  • 您删除应用
  • 数据超过保留期限（自动清理超过90天的旧数据）

删除数据后，我们会在30天内从所有系统中完全删除。

PlayTimer+ 承诺：

Privacy Policy

Last Updated: October 15, 2025

Your Privacy is Our Priority

PlayTimer+ ("we" or "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information.

1. Information We Collect

1.1 Automatically Collected Anonymous Information

To improve app functionality and user experience, we may collect the following completely anonymous information:

  • Device model (e.g., iPhone 14 Pro)
  • Operating system version (e.g., iOS 17.0)
  • App version number
  • Feature usage statistics (e.g., how many time limits created, which features used)
  • App crash and error reports

1.2 User-Provided Configuration Information

When you use app features, we may collect:

  • Child age group (e.g., 0-3 years, 3-6 years, not exact age)
  • Feature configurations (e.g., time limit settings, time slot configurations)
  • Purchase status (whether premium user)

1.3 We Do NOT Collect

✗ Your name, phone number, or email address

✗ Child's name, photos, or any personally identifiable information

✗ Precise geographic location coordinates

✗ Specific in-app content, messages, or chat logs

✗ Device unique identifiers (IDFA/UDID) for tracking

2. How We Use This Information

2.1 We Use Collected Information To:

  • Improve app features and user experience
  • Analyze which features are most popular for prioritization
  • Discover and fix bugs
  • Optimize app performance and stability
  • Understand user needs and develop new features

2.2 We NEVER:

✗ Sell your data to third parties

✗ Use it for advertising or targeted marketing

✗ Share it with other companies (unless required by law)

✗ Use it for cross-app or cross-site tracking

3. Data Storage and Security

3.1 Data Storage Location

✓ All data is stored in your private iCloud space

✓ Uses Apple CloudKit service (compliant with Apple security standards)

✓ Only you can access this data

✓ Other users cannot see your data at all

3.2 Security Measures

We take the following measures to protect your data:

  • End-to-end encrypted storage using Apple iCloud
  • Private Database isolation (completely private data)
  • Anonymous user identifiers (CloudKit system ID, not real identity)
  • No collection of sensitive personal information
  • Regular audits of collected data types

You have the following rights and can exercise them at any time:

4.3 Right to Delete

✓ Delete all collected data

You can request deletion of all collected data (feature in development), or delete directly from your iCloud.

4.4 Right to Withdraw Consent

✓ Withdraw previous consent anytime

You can turn off data collection in settings anytime, and we'll immediately stop collecting.
